#  Cinema Lab

Cinema Lab is a Flutter movie app that lets users search for movies and explore trending films using a real-time API.

---

##  Features
- Search movies instantly  
- View trending films  
- Movie details page  
- Clean UI  
- Smooth navigation 

---

##  Tech Stack
- Flutter  
- Dart  
- Bloc/Cubit  
- REST API (Dio)
